এই কোডের কমপ্লেক্সিটি কত? #shorts


complexity.cpp
                  
int myAlgorithm6(char *s){
int c=0;
for(int i=0 ; i < strlen(s) ; i++){
    if(s[i]=='a') 
        c++;
    }
        return c;
}


s স্ট্রিং এর দৈর্ঘ্য |s| হলে এখানে কমপ্লেক্সিটি হলো O(|s|^2) । কেন স্কয়ার হলো? কারণ strlen(s) ফাংশনের নিজের কমপ্লেক্সিটি হলো O(|s|) ,একে লুপের মধ্যে আরো O(|s|) বার কল করা হয়েছে। তাই strlen(s) এর মান আগে অন্য একটি ভ্যারিয়েবলের রেখে তারপর সেটা দিয়ে লুপ চালাতে হবে,তাহলে O(|s|) এ লুপ চলবে।


complexity.cpp
                  
//Complexity O(n)
int myAlgorithm6(char *s){
int c=0;
int len = strlen(s);
for(int i=0 ; i < len ; i++){
    if(s[i]=='a') 
        c++;
    }
        return c;
}
